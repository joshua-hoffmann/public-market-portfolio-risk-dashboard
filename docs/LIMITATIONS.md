# Limitations

This project is a descriptive historical analysis. It is not investment advice and does not provide buy, sell, hold, allocation, optimization, or forecasting recommendations.

## Data-source limitations

The project uses yfinance to access Yahoo Finance adjusted historical price data. This is practical for reproducible public-market analysis, but it is not treated as institutionally validated or investment-grade data.

Data availability, adjustment methods, vendor corrections, ticker history, missing observations, or API behavior may affect results. Users should independently validate data before using any analysis for high-stakes decisions.

Raw downloaded market data is not redistributed as freely licensed project data.

## Historical-analysis limitations

Historical volatility, drawdowns, return distributions, and correlations describe the selected historical period only. They do not predict future performance, future risk, future correlations, or future diversification outcomes.

## Asset-universe limitations

The selected ETFs and public-market assets are a methodological sample. They are not a recommendation list, not a preferred universe, and not a claim about which ETFs or assets are better.

## Metric limitations

Volatility, drawdown, return distribution, and correlation metrics each describe only part of historical risk behavior. They should be read together and with attention to the selected date range, data source, and assumptions.

## Chart limitations

Charts are intended to support interpretation of historical behavior. They are not rankings, recommendations, forecasts, or investment signals.

## Model limitations

This project does not validate an investment model, risk model, forecasting model, or portfolio optimization method.
